# Key Sprint

Typing speed test application built using React + Vite. It allows users to test their typing speed and accuracy by typing a given passage within a specified time limit.

## Install dependencies

`yarn install`

## Start Server

`yarn run dev`

## Node verison

`21.7.3`

## Host

`http://localhost:5173/`

## Production
